Vite로 빠른 개발 환경 구축

2024년 9월 21일 | 김동주 | 10분 읽기

🎯 개요

Vite는 ES 모듈을 기반으로 한 빠른 빌드 도구로, 개발 서버 시작 시간과 HMR 속도를 획기적으로 개선했습니다.

📚 Vite 시작하기

프로젝트 생성

npm create vite@latest my-react-app -- --template react-ts

기본 설정

// vite.config.js
export default defineConfig({
  plugins: [react()],
  server: { port: 3000 }
})

🔧 고급 설정

환경 변수와 플러그인을 활용하여 개발 환경을 최적화할 수 있습니다.

📝 마무리

Vite는 현대적인 프론트엔드 개발의 새로운 표준으로 자리잡고 있습니다.
