🎯 개요

Tailwind CSS는 유틸리티 우선 CSS 프레임워크로, 빠른 개발과 일관된 디자인 시스템 구축을 가능하게 합니다.

📚 기본 설정

tailwind.config.js 커스터마이징

module.exports = {
  theme: {
    extend: {
      colors: { primary: '#3b82f6' }
    }
  }
}

컴포넌트 클래스 생성

@layer components {
  .btn-primary { @apply px-4 py-2 bg-blue-500; }
}

🔧 실무 활용 팁

반응형 디자인과 다크 모드 지원으로 현대적인 UI를 구축할 수 있습니다.

📝 마무리

Tailwind CSS는 초기 학습 곡선이 있지만, 익숙해지면 매우 빠르고 일관된 UI 개발이 가능합니다.
