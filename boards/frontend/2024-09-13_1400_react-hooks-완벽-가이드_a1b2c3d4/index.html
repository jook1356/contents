🎯 개요

React Hooks는 React 16.8에서 도입된 혁신적인 기능으로, 함수형 컴포넌트에서도 상태 관리와 생명주기 메서드를 사용할 수 있게 해줍니다.

📚 기본 Hooks

1. useState - 상태 관리

const [count, setCount] = useState(0);

2. useEffect - 사이드 이펙트 처리

useEffect(() => {
  // 사이드 이펙트 로직
}, [dependencies]);

🔧 고급 활용법

커스텀 Hook을 만들어 로직을 재사용할 수 있습니다.

📝 마무리

React Hooks를 마스터하면 더 깔끔하고 재사용 가능한 코드를 작성할 수 있습니다.
