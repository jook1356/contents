🎯 개요

Jest와 React Testing Library는 React 애플리케이션의 테스팅을 위한 최고의 조합입니다.

📚 기본 설정

설치 및 설정

npm install --save-dev @testing-library/react

기본 테스트 작성

test('버튼이 올바르게 렌더링된다', () => {
  render(<Button>클릭하세요</Button>);
  expect(screen.getByRole('button')).toBeInTheDocument();
});

🔧 기본 테스트 작성

컴포넌트 테스트부터 커스텀 훅 테스트까지 다양한 패턴을 활용할 수 있습니다.

📝 마무리

좋은 테스트는 코드의 안정성을 보장하고 리팩토링을 안전하게 만들어줍니다.
